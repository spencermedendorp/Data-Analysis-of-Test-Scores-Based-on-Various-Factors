## Projects

### 1. 🏈 NFL Score Prediction Model
**Tools:** Python (Pandas, Scikit-learn, Matplotlib, NumPy)  
**Data:** Kaggle — Weekly NFL Team Offensive & Defensive Stats (2012–present)

Using weekly offensive and defensive team statistics sourced from Kaggle, this project builds a predictive model for NFL game scores. Features include passing yards, rushing yards, third down conversion rates, turnovers, and rolling season averages. The model explores which offensive and defensive factors are most predictive of points scored and allowed on a week-to-week basis.

📁 [View Project](./nfl-score-prediction)

### 2. 📊 Analysis of Student Test Scores Across Socioeconomic Factors
**Tools:** R (ggplot2, dplyr), Python  
**Data:** Public student performance dataset

This project investigates how factors such as parental education level, lunch type (a proxy for socioeconomic status), and test preparation courses relate to student performance across math, reading, and writing scores. Includes exploratory data analysis, visualizations, and regression modeling to quantify the impact of each variable.

📁 [View Project](./test-scores-analysis)
